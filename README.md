## 공지
- RUKA는 하나의 프로젝트로 계획되었으나, ORCA의 편의성 패치를 여러면에서 다양하게 진행하고자 하여, Project RUKA라는 이름으로 재탄생 시키려고 합니다. Project RUKA는 ORCA 알람 프로그램, 코드 Editer, 파일 정리 프로그렘, 명령어 편의성 패치 등으로 구성되어 있을 계획이며, 추후 변경되거나 수정될 수 있음을 명백히 밝힙니다.

- Project RUKA는 한과영 학생을 위해 설계한 것도 맞으나, 한편으로는 계산화학을 공부하는 모든 학생이 편하게 연구를 할 수 있으면 하는 마음으로 시작한 프로젝트입니다. 그러니 마음 편하게 사용해주시면 감사하겠습니다.

- 오픈소스이나, 상업적 이용을 자제해주세요.
